== Montando ambiente de execução ==
1 - Deve-se criar uma base de dados e atualizar a connection string no arquivo web.config;
2 - Rodar os comandos contidos no arquivo script.sql para criação das tabelas;
3 - Executar o programa.

== Códigos fontes == 
	O projeto foi commitado no GitHub no link: https://github.com/cadaviz/TesteAxado
	
== Tecnologias envolvidas ==
	- Foi utilizada a estrutura DDD para elaboração do projeto;
	- A injeção de dependências foi realizada utilizando o Ninject;
	- O banco foi construído utilizando o Migrations no modelo code-first e foi utilizada a versão 6 do Entity Framework;
	- Para a tradução de objetos de domínio para viewModel foi utilizado o AutoMapper;
	- A camada de visualização foi construida com MVC5;
	- Para modelagem das views foi utilizado o Razor;
	- Foram utilizadas rotas para deixar as urls mais amigáveis;
	- O site foi construído de forma responsiva utilizando o bootstrap;
	- Toda a aplicação foi desenvolvida no Visual Studio 2015 Enterprise.
